El fichero a compilar es Principal.c

DESARROLLO
===========

Para compilar en modo desarrollo, 
abrir el proyecto desarrollo.dcp.
File > Project > Open ...  [c:\user\corplus\desarrollo.dcp]

Para evitar grabar a flash muchas veces:
Compile > Compile to Target > Compile to RAM


GENERACION FINAL
================

Para generar el binario final lo mas limpio y compacto posible, 
utilizar el proyecto rcm2110.dcp
File > Project > Open ...  [c:\user\corplus\rcm2110.dcp]

Compilar con
Compile > Compile to .bin File > Compile to Flash
